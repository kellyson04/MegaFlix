# MegaFlix
**Data de início do projeto:** 17/03/2026

Projeto criado para acompanhar minha evolução no Spring Boot.

Comecei a estudar Spring há pouco tempo e, depois de fazer um CRUD mais básico, criei esse projeto para continuar praticando e subir de nível no backend com Java.

A ideia é evoluir esse sistema aos poucos, saindo do básico e chegando em temas como Spring Security, autenticação e uso de tokens.
